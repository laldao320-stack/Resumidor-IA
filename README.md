# Streamlit Text Summarizer

A minimal Streamlit app with a text input and a **Summarize** button.

## Run

Prereq: install **Python 3.10+** and ensure `python` works in PowerShell.

```bash
cd "C:\Users\LU\streamlit-summarizer"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
